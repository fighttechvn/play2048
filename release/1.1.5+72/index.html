What's new in 72

  • Beat music game, reimagined — Piano-Tiles tap: tap each tile in its own lane (no bottom hit-line). Die if a tile slides off un-tapped or you tap the wrong lane.
  • Endless is the default — the beat speeds up over time the longer you last; levels (each a different song) moved to a By-level picker.
  • New first-time tutorial — a Path-style inline coachmark guides you with a glowing fingertip on the live game (the old "How to play" / "Start practice" screen is gone).
  • Carries forward 1.1.5(70): cross-device daily streaks, Android leaderboard & sign-in fixes, smoother tab bar, newest games first.

Verification

  • Beat redesign — verified in the browser harness (?dbg=1): welcome → guided wait/tap → finish → accelerating endless (speed 293→351 over 5s), survive-by-tapping, tile-falls-off death, wrong-lane death, level clear, full menu navigation. No console errors.
  • 1.1.5 device E2E (streaks, favorites, settings, name-on-home, leaderboard) — see the 🧪 E2E report from build 70 (unchanged UI).
  • Pending: Android leaderboard/sign-in not re-verified on device (only Android 4.3 hardware available; app needs 6.0+).

Included changes

  • feat(beat) — Piano-Tiles tap mechanic, Endless-default, time-based speed ramp, Path-style inline tutorial; removed bottom hit-line / danger bar / hold tiles / how-to-play screen.
