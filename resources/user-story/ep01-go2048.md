# EP01: go2048 Game User Stories

The classic 2048 number puzzle, rebuilt as an H5 game (TypeScript + PixiJS v7),
packaged for Web, iOS 12+, and Android 5.1+ with Capacitor 6.

## EP01.US001: Play a move
As a player, I want to slide the tiles in a direction so that matching numbers
merge and I make progress toward 2048.

Acceptance criteria:
- Arrow keys / WASD (desktop) and swipe up·down·left·right (touch) move every tile.
- Two adjacent tiles with the same value merge into one of double the value.
- Each successful move spawns one new tile (2 or 4) in a random empty cell.
- A move that changes nothing spawns no tile and is ignored.
- Tiles animate (slide + merge-pop) at 60 fps.

## EP01.US002: Win and keep going
As a player, I want to be told when I reach the 2048 tile and be able to keep
playing for a higher score.

Acceptance criteria:
- Reaching a 2048 tile shows a "You win" overlay with a "Keep going" action.
- Choosing "Keep going" dismisses the overlay; the board continues.
- The win overlay is shown only once per game.

## EP01.US003: Game over
As a player, I want to know when no moves remain so that I can start again.

Acceptance criteria:
- When the board is full and no merge is possible, a "Game over" overlay appears.
- The overlay offers "Try again" which starts a fresh game.

## EP01.US004: Score and best
As a player, I want my score and best score tracked so I can beat my record.

Acceptance criteria:
- Score increases by the value of each merged tile.
- Best score persists across sessions and only ever increases.

## EP01.US005: Resume my game
As a player, I want my board restored when I reopen the app.

Acceptance criteria:
- The current grid, score, best, and win-state are saved after every move.
- On launch the saved game is restored; a finished, unwinnable board starts fresh.
- Storage works on web and native (`@capacitor/preferences`).

## EP01.US006: Theme
As a player, I want Dark / Light / System themes so the game is comfortable.

Acceptance criteria:
- A theme toggle cycles System → Light → Dark; the choice persists.
- "System" follows the OS color scheme; the document chrome (page bg, status bar)
  matches the theme.

## EP01.US007: Language
As a player, I want the interface in my language.

Acceptance criteria:
- 5 locales: English, Tiếng Việt, 한국어, 中文, العربية.
- A language toggle cycles locales; the choice persists.
- Arabic renders right-to-left (header mirrored).
