How to play

Each game is simple to start and hard to put down.

1

go2048 — merge

Swipe to slide all tiles. Two equal numbers merge into double. Climb to 2048 and beyond.

2

Zip — connect

Trace one continuous path through every cell, passing the numbered dots from 1 to N in order.

3

Patch — divide

Cut the grid into rectangles so each contains a single number equal to its area.
